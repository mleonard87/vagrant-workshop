What is Vagrant?

  • Provisioning tool
  • Uses VirtualBox, Hyper-V, Docker + VMWare, AWS, others.
  • Salt!
  • Shares folders, forwards ports, syncs files, etc.
  • Isolation of dependencies - even a database as a dependencies
  • "works on my machine"
  • Vagrantfile goes in repo

Provisioning Your First(?) Box


mkdir vagrant-test
cd vagrant-test

vagrant init ubuntu/xenial64
vagrant up

vagrant ssh

vagrant snapshot save my-first-snapshot


vagrant halt

vagrant destroy

Workshop Vagrantfile


Vagrant.configure("2") do |config|

  config.vm.box = "ubuntu/xenial64"

  config.vm.network "forwarded_port", guest: 80, host: 18000

  config.vm.provision "shell", path: "scripts/provision.sh"

  config.vm.provider "virtualbox" do |v|
    v.memory = 1024
    v.cpus = 2
  end

end
